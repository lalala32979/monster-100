# 物体编辑器导入导出使用说明
* 这个压缩包中包含由英雄三国编辑器物体编辑器导出的数据信息。
* 这些数据中，有一部分是可读可写的，你可以在本文件的同级文件夹下找到被命名为data.xlsx的表格文件对数据进行批量的操作。
* 若你解压并修改了数据，重新压缩时注意本文件应处于压缩包的第一级，不要增加上级文件夹。
# 文件结构
## 数据关联的文件夹存储的数据对应关系如下：
+ editor_unit――单位
+ ability_all――技能
+ editor_decoration――装饰物
+ editor_destructible――可破坏物
+ editor_item――物品
+ technology_all――科技
+ store_all――平台道具
+ modifier_all――魔法效果
+ projectile_all――投射物
+ sound_all――音效
## 上述文件夹的data.xlsx中包含了你可以批量修改的全部内容。对应实体的属性可能分布在表中的各个sheet，可以对照后进行修改。
## 在修改时请注意，每个sheet的前两列及“meta”sheet请不要进行手动修改，若要修改名称列，将以“base_info”sheet中的数据为准。
## 除上述文件夹的data.xlsx外，其余文件中存储的是物体编辑器条目依赖的自定义资源或属性定义，请不要手动更改里面的内容，避免造成不必要的损失。
## 如果编辑器版本改变，导出包不一定能够正常导入，建议重新导出后导入。
